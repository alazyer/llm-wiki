# Forward Deployed Engineering 101

## Summary
Forward Deployed Engineers find high-value AI use cases inside real customer workflows, design evals around actual human steps, and deploy the smallest reliable autonomy over existing systems. The role blends consulting, product, engineering, and operations.

## Key Takeaways
- FDEs begin by observing where time, judgment, handoffs, and repeated decisions live.
- Not every problem needs an agent; sometimes code, workflow change, or manual process is better.
- Evals should trace real human work and use golden examples.
- Deploy over existing systems through APIs before replacing them.
- Start with the smallest autonomy that creates value and can be verified.
- A 30-day ramp should cover tools, loops, guardrails, structured output, checkpointing, retries, cost, and evals.
