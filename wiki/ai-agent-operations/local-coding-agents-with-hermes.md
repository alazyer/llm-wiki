# Local Coding Agents with Hermes

## Summary
Local coding agents are best treated as cheap junior workers for narrow, low-risk cards. Hermes can route work through a Kanban board, assigning simple tasks to local models and escalating complex or risky tasks to stronger agents.

## Key Takeaways
- Local models should complement Claude/Codex, not replace them wholesale.
- Small cards beat broad goals; narrow scope makes weaker models useful.
- A Kanban board is the durable task bus and audit trail.
- Routing rules in `AGENTS.md` decide which worker handles which class of card.
- Local output needs stronger review, tests, and blast-radius limits.
- The best economic use is high-volume, low-risk work with clear verification.
