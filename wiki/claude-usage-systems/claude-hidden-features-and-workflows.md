# Claude Hidden Features and Workflows

## Summary
Claude’s leverage comes from combining product features: Projects, Artifacts, Memory, Extended/Adaptive Thinking, scheduled tasks, skills, browser/context integrations, Claude Code, and API prompt caching. Most users underuse the surrounding workflow features.

## Key Takeaways
- Projects keep domain context attached to a body of work.
- Artifacts turn conversations into editable documents, apps, and prototypes.
- Memory and custom instructions reduce repeated setup.
- Thinking modes help with complex reasoning when speed is less important than depth.
- Scheduled tasks and skills move Claude from chat to recurring workflow support.
- Claude Code and `CLAUDE.md` bring project-specific behavior into software work.
- API prompt caching can reduce repeated-context costs for developers.
