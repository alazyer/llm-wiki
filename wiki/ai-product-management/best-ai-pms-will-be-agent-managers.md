# Best AI PMs Will Be Agent Managers

## Summary
Managing agents looks less like writing perfect prompts and more like managing people: assign roles, set principles, observe output, give specific feedback, and turn corrections into durable operating memory.

## Key Takeaways
- Agent teams are a management problem as much as a technical problem.
- Principles outperform over-specified procedures for ambiguous work.
- Do not quit during weak first outputs; onboarding is part of the system.
- Different agent roles need different management styles: creative agents need room, compliance agents need strict gates.
- Corrections should become files, examples, evals, or skills.
- PM strengths map well to agent scoping, context curation, coordination, and feedback loops.
