# AI Agents Overview

## Summary
AI agents sit on a spectrum from chatbots to tool-using workflows to autonomous systems. A useful agent needs tools, memory, goals, loops, and boundaries; a practical beginner build can start as a Telegram bot backed by Claude and grow skills over time.

## Key Takeaways
- Agent spectrum: chat, tool use, workflows, and autonomous agents.
- Agents need tools to act, memory to persist, loops to continue, and guardrails to stay safe.
- Common agent types include research, writing, coding, business operations, and planning agents.
- A Telegram bot is a simple control surface for a personal agent.
- Useful extensions include web search, notes, private access control, cost tracking, and daily briefings.
- Checkpointing and memory prompts help agents become more consistent over time.
